Recurring events generator for Totara

Since you are on this page, you might already know that generating recurring events in Totara can be a pain, isn't? It's doable if you need just a small number of events, but if for example, you have a training session that is running twice a week every Monday and Wednesday for 12 month, it can take a while to create 52 weeks X 2 events/week = 104 events.

That's when this tool comes handy.

Generate recurring events

Step 1

Upload rooms
(optional)

Simply generate rooms report from your Totara instance to add rooms to your recurring events.

You can skip this step if you want to create sessions without rooms assigned.

Go to this step

Step 2

Add custom fields
(optional)

The easiest way to get your custom fields is to generate a backup of the Seminar activity with at least one event. One you import the backup in this tool you will be able to enter data in custom fields.

This step is optional and you can skip it if you don't have any custom fields or if you want to leave custom fields empty.

Go to this step

Step 3

Generate events
:)

Create as many recurring events as you like, download the Seminar activity backup and upload it to your Totara instance. That's it!

Go to this step

Limitations of this tool

  1. You can generate only events with one session. If there's a need I'm happy to add functionality to generate multiple sessions for one event. Email me at olexiy.dev@gmail.com.
  2. Only "text" type custom fields are supported. And again, if there's a need to include other field types, I'm happy to work on it. Email to your requests to olexiy.dev@gmail.com.
  3. I have tested this on Totara 12, but I assume it should work on the other versions starting from 9. I suggest you to always check on your test environment before the production one.
